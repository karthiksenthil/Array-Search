PROGRAM TO SEARCH FOR ELEMENT IN ARRAY

This program primarily implements the array search using BINARY SEARCH algorithm. The prerequisites of this algorithm is a sorted array, which is implemented using BUBBLE SORT. So the program takes the array input from user,sorts it and takes the input for the element to be searched. The program uses two method to implement this: sort_arr and binary_search.
